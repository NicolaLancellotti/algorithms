# Algorithms & Data Structures  

## Algorithms

### Search
- sequential search O(n)
- binary search O(log(n))

### Sort
- selection sort Θ(n^2)
- insertion sort O(n^2)
- bubble sort Θ(n^2)
- heap sort O(nlog(n))
- merge sort Θ(nlog(n))

### Other
- fibonacci1 O(n)
- fibonacci2 O(log(n))
