# Algorithms & Data Structures  

## Algorithms

### Search
- sequential search O(n)
- binary search O(log(n))

### Sort
- selection sort Θ(n^2)
- insertion sort O(n^2)
- bubble sort Θ(n^2)
- heap sort O(n log(n))
- merge sort Θ(n log(n))
- quick sort O(n^2)
- integer sort O(n*k)  where k = max value - min value + 1

### Other
- fibonacci1 O(n)
- fibonacci2 O(log(n))
